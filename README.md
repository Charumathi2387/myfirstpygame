Guess the Word - Python Game 


## How to Play  
1️ The game picks a random word.  
2️ You guess letters one by one.  
3️ If the letter is correct, it appears in the word.  
4️ If wrong, you lose a chance.  
5️ Guess the word before running out of attempts!  

 Technologies Used  
  Python  
  Random module  
  Loops & String manipulation
